Imagen según la resolución del dispositivo

Imágenes

Si queremos una imagen adaptable basta con usar width:100% y height: auto:

Si se quiere una imagen con una dimensión y posición fija se debe usar object fit y object position:

Contain

La imagen se adapta según sus dimensiones.

Cover

La imagen se adapta para cubrir el contenedor

fill

La imagen rellena el contenedor.

None

La imagen guarda sus proporciones

La propiedad object position determinará la posición de la imagen dentro de su contenedor. Se le pueden pasar como valores medidas o "top" "bottom" "left" "right" o "center".

Mácaras

Determinan el área visible de la imagen y la forma que puede tomar.
Se determina con la propiedad clip-path, a la que se le pueden pasar varias funciones como valor:

Circle

Elipsis

Inset

Poligon
Puedes utilizar la página Css clip-path Maker para generar el valor de los nodos
