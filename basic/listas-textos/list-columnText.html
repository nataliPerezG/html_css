FUENTES

Adicional a lo que conoces:

  • "font-variant: small-caps": capitaliza el texto escrito en minúsculas.
  • "line-height" asigna la altura de línea, es decir, el valor del interlineado.
  • Font-display: si tenemos una fuente personalizada, la propiedad font-display le indica al navegador cómo cargar el texto mientras la fuente personalizada carga.
      Valores:
    • Block: oculta el texto hasta que la fuente se descarge.
    • Swap: utiliza una fuente por defecto hasta que la fuente cargue.
  • text-indent: proporciona sangría a la primera línea del párrafo.
  • text-overflow: define cómo se manejará el texto cuando excede al contenedor. Su valor puede ser o ellipsis(pone ...) PERO para que funcione, el contenedor debe tener asignado "overflow:hidden", "white-space:nowrap". El texto quedaría así:

    • Había una vez un barco pequeñito que no podía navegar
